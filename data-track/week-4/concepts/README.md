# Week 4 — Concept Scripts

Each file in this folder is a runnable Python script that mirrors one chapter from the Week 4 curriculum. Run them locally to follow along with the examples without copy-pasting from the browser.

| File | Chapter |
|---|---|
| `companion_ch1_intro_pandas_dataframes.py` | Ch1: DataFrames and Inspection |
| `companion_ch2_selecting_filtering_sorting.py` | Ch2: Selecting, Filtering, and Sorting |
| `companion_ch3_grouping_aggregation.py` | Ch3: Grouping and Aggregation |
| `companion_ch4_joining_merging.py` | Ch4: Joining and Merging DataFrames |
| `companion_ch5_strings_dates.py` | Ch5: Working with Strings and Dates |
| `companion_ch6_advanced_transformations.py` | Ch6: Advanced Transformations |
| `companion_ch7_writing_results.py` | Ch7: Writing Data |
| `companion_ch8_visualizing_data.py` | Ch8: Visualizing Data with Pandas |
| `companion_ch9_pandas_alternatives.py` | Ch9: Alternatives to Pandas |

## Run a script

```bash
cd data-track/week-4
pip install -r requirements.txt
python concepts/companion_ch1_intro_pandas_dataframes.py
```
