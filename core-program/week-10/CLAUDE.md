# CLAUDE.md

This is a HackYourFuture educational repo. Our students are referred to as "trainees". Avoid the use of the word "student".

Documentation in markdown file with a name that starts with "README" is trainee-facing; don't expose internal/admin details.

Other markdown documentation files are intended to instructors. These instructor are also referred to as "teachers" or "mentors".

All code should work cross-platform on MacOS, Windows and Linux.
