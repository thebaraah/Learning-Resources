# Git CLI exercise
1. Find a file called 'message.txt', read its contents
2. Replace the message with the text "CLI FIXED ✅"
3. Clean up by deleting two old log files
4. Find out what does the cow say

## Restrictions
- You may only use the CLI
- Use only relative paths

## Hints
- Use `tab` to autocomplete file names
- Don't forget to use `pwd` to find out where you are
